# Bloom & Veg (starter theme)

A lightweight starter theme with WooCommerce support.

## Features

- Basic templates: `index.php`, `single.php`, `page.php`, `archive.php`, `search.php`, `404.php`
- WooCommerce wrapper: `woocommerce.php`
- Minimal responsive layout and product grid styling
